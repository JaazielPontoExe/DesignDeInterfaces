HTML

Nesta seção, exploramos os fundamentos do HTML, destacando como essa linguagem pode ser utilizada para estruturar páginas web de forma eficiente.

Aulas

Introdução ao HTML e Ferramentas

Apresentação da disciplina e ferramentas como VS Code, GitHub e W3C Validator. Introdução às tags principais do HTML como <html>, <head>, <body>, <p> e <ul>.

Imagens, Links e Formulários

Exploração do uso de <img> com caminhos relativos, links com <a>, e criação de formulários com <input>, <select>, <textarea> e <button>.

CSS

Nesta seção, abordamos os fundamentos do CSS, a linguagem de estilo utilizada para definir a aparência das páginas web.

Aulas

Seletores, Texto e Box Model

Introdução ao uso de CSS em páginas HTML e estudo de seletores, formatação de texto e Box Model.

Posicionamento de Elementos

Estudo sobre display, position, z-index, overflow e organização de elementos.

Flexbox e Responsividade

Introdução ao modelo Flexbox com flex-direction, justify-content, align-items, entre outros.
