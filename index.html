HTML

Nesta seção, exploramos os fundamentos do HTML, destacando como essa linguagem de marcação é essencial para a construção da estrutura básica de páginas web. O HTML fornece a base sobre a qual outros componentes, como CSS e JavaScript, são aplicados para dar vida e funcionalidade às páginas. Vamos abordar as principais tags e elementos HTML, como a definição de um documento HTML, o cabeçalho da página, o corpo do conteúdo, links, imagens e listas, entre outros, para que você compreenda como organizar e estruturar um conteúdo de maneira eficiente e semântica.

Aulas

Introdução ao HTML e Ferramentas

Nesta aula introdutória à disciplina de Desenvolvimento Web, são abordados conceitos fundamentais sobre o HTML e suas ferramentas essenciais. O uso de ferramentas como o VS Code, um editor de código eficiente, o GitHub, plataforma para versionamento e colaboração, e o W3C Validator, que valida o código HTML de acordo com os padrões da web, é explicado. Além disso, são apresentadas as principais tags do HTML, incluindo <html>, para definir o início do documento HTML, <head>, para informações meta e de configuração, <body>, que contém o conteúdo da página, <p>, para parágrafos de texto, e <ul>, para listas não ordenadas.

Imagens, Links e Formulários

Esta seção explora o uso de imagens, links e formulários em HTML. A tag <img> é abordada para a inclusão de imagens, com destaque para a utilização de caminhos relativos. A tag <a> é explicada para criar links de navegação entre páginas. Também são detalhados os elementos essenciais para a criação de formulários, como o <input>, que permite entradas de dados, o <select>, para menus suspensos, o <textarea>, para áreas de texto, e o <button>, para botões interativos dentro de um formulário.

CSS

Nesta seção, abordamos os fundamentos do CSS, a linguagem de estilo amplamente utilizada para definir a aparência visual das páginas web. O CSS permite controlar o layout, as cores, as fontes, os espaçamentos e outros aspectos visuais, separando a estrutura (HTML) da apresentação. Vamos examinar os conceitos principais do CSS, incluindo seletores, propriedades e valores, além de aprender sobre o modelo de caixa (Box Model) e o controle do posicionamento de elementos, o que é essencial para criar designs atraentes e responsivos que funcionem bem em diferentes dispositivos e tamanhos de tela.

Aulas

Seletores, Texto e Box Model

Esta aula oferece uma introdução ao uso de CSS para estilização de páginas HTML. São abordados os principais seletores, que permitem a aplicação de estilos específicos a elementos, classes e IDs. Também é discutida a formatação de texto, incluindo o controle de fontes, tamanhos, espaçamento e alinhamento. Além disso, é explicado o conceito de Box Model, fundamental para entender como os elementos são exibidos, incluindo o modelo de caixas com propriedades como padding, border, margin e o conteúdo do próprio elemento.

Posicionamento de Elementos

Esta seção trata do posicionamento de elementos na página usando CSS. São exploradas as propriedades display, que define o tipo de exibição de um elemento (como block, inline, etc.), e position, que controla o posicionamento dos elementos na tela, com valores como static, relative, absolute e fixed. Também são abordadas as propriedades z-index, que define a sobreposição de elementos, e overflow, que gerencia o comportamento de conteúdo que ultrapassa os limites de um contêiner.

Flexbox e Responsividade

Esta aula introduz o modelo Flexbox, uma poderosa ferramenta para a criação de layouts flexíveis e responsivos. São abordadas as propriedades essenciais do Flexbox, como flex-direction, que define a direção dos itens dentro de um contêiner flexível, justify-content, que controla o alinhamento dos itens ao longo do eixo principal, e align-items, que gerencia o alinhamento ao longo do eixo transversal. A aula também explora como utilizar o Flexbox para criar designs responsivos, adaptáveis a diferentes tamanhos de tela.
